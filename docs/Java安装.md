## 自动安装

在使用 `PCL 2 启动器` 启动 Minecraft 时，会自动检测 Java 版本，如果不存在需要的版本，则会自动进行安装。

## 手动安装

前往 `java.com` 或 `甲骨文官网` 或 `群文件` 下载 Java/Jdk 安装包，并进行安装。
